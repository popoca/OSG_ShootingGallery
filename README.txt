Para correr el archivo de Visual Studio 2010 se requiere:
- Tener instalado OSG por ahi. 
- Tener declaradas las siguientes env vars:
	- $(OSG_INCLUDE_PATH) - apunta al folder /include de tu instalacion OSG
	- $(OSG_LIB_PATH) - apunta al folder /lib de tu instalacion OSG
